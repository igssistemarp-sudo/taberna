PROMPT COMPLETO – MÓDULO DE CLIENTES E DELIVERY

Criar um módulo completo de CLIENTES E DELIVERY para o sistema IGS Lanchonete PRO, utilizando PostgreSQL com senha padrão 123.

OBJETIVO

O sistema deve ser extremamente rápido para atendimento telefônico, WhatsApp e pedidos online.

O atendente deve conseguir localizar um cliente em segundos apenas digitando o telefone.

=================================================================
PESQUISA RÁPIDA DE CLIENTE
=================================================================

Ao abrir um novo pedido Delivery, mostrar uma tela de pesquisa rápida.

Campo principal:

Telefone / WhatsApp

Máscara:
(00) 00000-0000

Ao digitar o telefone:

- Procurar automaticamente no banco
- Procurar por telefone ou WhatsApp
- Buscar enquanto digita
- Não exigir Enter

Se encontrar:

Exibir:

Nome do Cliente
Telefone
Bairro
Endereço
Número
Último Pedido
Data da Última Compra
Total Gasto

Botões:

Selecionar Cliente
Editar Cadastro
Ver Histórico

Se NÃO encontrar:

Exibir botão:

➕ Cadastrar Novo Cliente

BAIRRO COM TAXA DE ENTREGA
=================================================================

Tabela:

Campos:

- Nome
- Cidade
- Taxa de Entrega
- Tempo Médio
- Ativo

Exemplo:

Centro = R$ 5,00
Cidade Nobre = R$ 8,00
Bom Retiro = R$ 10,00
Cariru = R$ 7,00

Ao selecionar o bairro:

Buscar automaticamente:

- Taxa de entrega
- Tempo médio
